== Description ==

In a digital world, Statistics Website gives you insights on your website's visitors, your marketing campaigns and much more, so you can optimize your strategy and online experience of your visitors. Powered by Piwik. No need to install it on your server. Generates IDsite your webmasters.bumbablog.com website. No need to reload your server. Get statistics from your web page from a remote server. Multi site.

= Features =

- Real Time Visitor Count
- Last 30 Minutes
- Last 24 Hours
- Visits Over Time
- Visits by Server Time
- Visitor Map
- Visitor Location (Country)
- Returning Visits Over Time
- Pages per Visit
- Search Engines
- Referrer Websites
- Keywords
- And more...

== Installation ==

Statistics Website is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Copy el plugin from the downloaded and extracted file.

2. Paste it in wp-Content/plugins folder on your Wordpress Installation 

3. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated.
